## Setup
The following scripts can be used to initialize this directory with the relevant videos (as specified in [cvlab, vot2017]/list.txt):    
CVLab:
```
$ chmod +x cvlab/cvlab_setup.sh
$ cd cvlab
$ ./cvlab_setup.sh
```    
VOT2017:    
```
$ chmod +x vot2017_setup.sh
$ cd vot2017
$ ./vot2017_setup.sh
```
